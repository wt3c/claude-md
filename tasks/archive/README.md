# Archive - Tarefas Concluídas

Este diretório armazena tarefas que foram completadas e arquivadas.

## Quando Arquivar

Mova um arquivo `todo.md` para cá quando:
- A tarefa foi completamente concluída
- Todos os critérios de aceite foram atendidos
- O review foi preenchido
- Não há pendências relacionadas

## Naming Convention

Renomeie o arquivo ao mover para archive:

```
todo.md → YYYY-MM-DD_nome-da-tarefa.md
```

Exemplo:
```
2026-05-14_configuracao-inicial-claude-code.md
```

## Retenção

Arquivos aqui são mantidos para:
- Referência histórica
- Aprendizado (consultar decisões passadas)
- Auditoria

Não há prazo de exclusão automática.
